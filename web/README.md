## 🛠️ **本地开发**

### 环境要求
- Node.js 16+
- npm 或 yarn

### 启动开发服务器
```bash
cd web
npm install
npm start
```

访问 http://localhost:3000 查看应用

### 构建生产版本
```bash
npm run build
```

### 部署到GitHub Pages
```bash
npm run deploy
```
